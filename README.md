### ケハレ帖
日常の食事（ケ）に「小さなハレ」を加える習慣化Webアプリ

**Problem**
毎日の食事が単調になりがちで、献立を考えること自体が疲労の原因になっている

**Solution**
記録とポイント制で習慣化を支援し、選択式UIの献立相談機能で「何を食べるか」の決断疲れを解消

### KikakuHub
イベント企画の「テーマ決め」と「日程調整」を一括支援するWebアプリ

**Problem**
イベントの企画段階で、テーマの需要と参加者の都合をまとめる手段がなく、主催者の負担が大きい

**Solution**
テーマへの投票・参加可能時間の登録を一元管理し、スライディングウィンドウアルゴリズムで候補日時TOP3を自動提案
